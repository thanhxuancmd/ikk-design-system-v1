Usage Guidelines

권장사항

  • 어두운 배경에서 높은 대비를 위해 밝은 색상 사용
  • 게이밍 콘텐츠의 경우 네온 색상으로 강조
  • 카드 기반 레이아웃으로 콘텐츠 구조화
  • 모바일 반응형 디자인 적용
  • SOOP 오렌지를 주요 브랜드 컬러로 활용

금지사항

  • 밝은 배경색 사용 금지 (다크 테마 유지)
  • 과도한 애니메이션으로 시청 방해
  • 브랜드 컬러 팔레트 외의 색상 무분별 사용
  • 텍스트 가독성을 해치는 색상 조합
  • 게이밍 분위기와 맞지 않는 디자인 요소

접근성 가이드라인

색상 대비

  • • 일반 텍스트: 최소 4.5:1 대비
  • • 큰 텍스트: 최소 3:1 대비
  • • UI 요소: 최소 3:1 대비

키보드 내비게이션

  • • 모든 인터랙티브 요소 포커스 가능
  • • 명확한 포커스 표시
  • • 논리적인 탭 순서
